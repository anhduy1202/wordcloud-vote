# Wordcloud Vote ☁️
> Generate a word cloud from your votes

> Designed and made with 💙 by Daniel Truong 

## What's a word cloud❔
>  A word cloud is a collection, or cluster, of words depicted in different sizes, it looks something like this:

![image](https://user-images.githubusercontent.com/58461444/220288669-572b759d-4b0d-4810-b028-0959a00f2344.png)

# About WordCloud Vote 📝
> This is a poll web platform application that allows users to create their own poll and share it with others via links, results from voters then will
form a beautiful word cloud

# Tech Stacks 📡
- Next.js + TypeScript (full-stack framework)
- TailwindCSS (styling)
- Prsima (ORM)
- MongoDB (Database)
- FastAPI + Python (WordCloud API)
- Other packages (React Hook Form, Yup, HeadlessUI,...)
